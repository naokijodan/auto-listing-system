Task: Add Phase 3-B management API endpoints to apps/api/src/routes/products.ts

The Prisma schema already has these fields on Product model:
- tags: String[] @default([])
- memo: String? @db.Text
- purchaseShippingCost: Int?
- initialPurchasePrice: Int?

DO NOT touch prisma schema or run prisma commands. Only edit apps/api/src/routes/products.ts.

1. Add tag filter to GET '/' endpoint:
   - Read `tag` from req.query
   - If tag is provided, add `where.tags = { has: tag as string }` to the existing where object

2. Add PATCH /:id/tags endpoint (before export default router):
```typescript
router.patch('/:id/tags', async (req, res, next) => {
  try {
    const { id } = req.params;
    const { tags } = req.body;
    if (!Array.isArray(tags)) {
      return res.status(400).json({ error: 'tags must be an array of strings' });
    }
    const product = await prisma.product.update({
      where: { id },
      data: { tags },
    });
    res.json(product);
  } catch (error) {
    next(error);
  }
});
```

4. Add PATCH /:id/purchase-info endpoint:
```typescript
router.patch('/:id/purchase-info', async (req, res, next) => {
  try {
    const { id } = req.params;
    const { purchaseShippingCost, initialPurchasePrice } = req.body;
    const data: any = {};
    if (typeof purchaseShippingCost === 'number') data.purchaseShippingCost = purchaseShippingCost;
    if (typeof initialPurchasePrice === 'number') data.initialPurchasePrice = initialPurchasePrice;
    const product = await prisma.product.update({
      where: { id },
      data,
    });
    res.json(product);
  } catch (error) {
    next(error);
  }
});
```
