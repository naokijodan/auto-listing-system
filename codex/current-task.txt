Currently all buttons (削除, 一括停止, 再出品, 価格一括変更, エクスポート) have no onClick handlers despite backend APIs being fully implemented.

## Context

- File to modify: apps/web/src/app/listings/page.tsx
- Also need to add API methods in: apps/web/src/lib/api.ts
- Reference implementation: apps/web/src/app/joom/page.tsx (has working handlers)
- The page uses `selectedIds` (Set<string>) to track selected listings
- The page has `mutate` from SWR to refresh data after operations
- Import `postApi` and `deleteApi` from '@/lib/api' (already available)

## Backend APIs (all implemented and working)

1. **Bulk Delete**: `POST /api/listings/bulk/delete`
   - Body: `{ listingIds: string[], confirm: true }`
   - Constraint: Cannot delete ACTIVE listings (must pause first)
   - Response: `{ success: true, data: { deletedCount: number } }`

2. **Bulk Status Update**: `POST /api/listings/bulk/update-status`
   - Body: `{ listingIds: string[], status: 'DRAFT' | 'PAUSED' | 'ACTIVE' | 'ENDED' }`
   - Response: `{ success: true, data: { updatedCount: number } }`

3. **Bulk Publish**: `POST /api/listings/bulk/publish`
   - Body: `{ listingIds: string[] }`
   - Only works on DRAFT/ERROR listings
   - Response: `{ success: true, data: { queuedCount: number } }`

4. **Bulk Price Update**: `POST /api/listings/bulk/update-price`
   - Body: `{ listingIds: string[], adjustment: { type: 'percent' | 'amount' | 'fixed', value: number } }`

### 2. Add handler functions (inside the component, before the return statement)

```typescript
const [isProcessing, setIsProcessing] = useState(false);

const handleBulkDelete = useCallback(async () => {
  if (selectedIds.size === 0) return;
  if (!window.confirm(`${selectedIds.size}件の出品を削除しますか？アクティブな出品は先に停止してください。`)) return;
  setIsProcessing(true);
  try {
    await postApi('/api/listings/bulk/delete', {
      listingIds: Array.from(selectedIds),
      confirm: true,
    });
    setSelectedIds(new Set());
    mutate();
  } catch (error: any) {
    alert(error.message || '削除に失敗しました');
  } finally {
    setIsProcessing(false);
  }
}, [selectedIds, mutate]);

const handleBulkPause = useCallback(async () => {
  if (selectedIds.size === 0) return;
  setIsProcessing(true);
  try {
    await postApi('/api/listings/bulk/update-status', {
      listingIds: Array.from(selectedIds),
      status: 'PAUSED',
    });
    setSelectedIds(new Set());
    mutate();
  } catch (error: any) {
    alert(error.message || '一括停止に失敗しました');
  } finally {
    setIsProcessing(false);
  }
}, [selectedIds, mutate]);

const handleBulkPublish = useCallback(async () => {
  if (selectedIds.size === 0) return;
  setIsProcessing(true);
  try {
    await postApi('/api/listings/bulk/publish', {
      listingIds: Array.from(selectedIds),
    });
    setSelectedIds(new Set());
    mutate();
  } catch (error: any) {
    alert(error.message || '再出品に失敗しました');
  } finally {
    setIsProcessing(false);
  }
}, [selectedIds, mutate]);

const handleExport = useCallback(() => {
  if (selectedIds.size === 0) return;
  const selected = listings.filter(l => selectedIds.has(l.id));
  const csv = [
    ['ID', 'Title', 'Marketplace', 'Status', 'Price', 'Currency'].join(','),
    ...selected.map(l => [
      l.id,
      `"${(l.product?.title || '').replace(/"/g, '""')}"`,
      l.marketplace,
      l.status,
      l.listingPrice,
      l.currency,
    ].join(','))
  ].join('\n');
  const blob = new Blob([csv], { type: 'text/csv' });
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = `listings-export-${new Date().toISOString().slice(0,10)}.csv`;
  a.click();
  URL.revokeObjectURL(url);
}, [selectedIds, listings]);
```
