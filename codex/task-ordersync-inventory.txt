## Task: order-sync定期同期にlisting検索 + InventoryEvent + Product.status='SOLD' 追加

### 対象ファイル
`apps/worker/src/processors/order-sync.ts`

### 背景
定期的な注文同期（4時間ごと）で新規注文を作成した際、InventoryEventの記録とProduct.status='SOLD'の更新が行われていない。
Shopifyのwebhook処理（`apps/worker/src/lib/shopify-webhook-processor.ts` 行141-155）のパターンに従い、注文同期時にも同様の処理を追加する。

### 現在のコード（行359-393）
新規注文作成後、売上明細(Sale)を作成するforループがあるが、listing検索もInventoryEvent作成もProduct.status更新もない。

```typescript
// 新規注文を作成
const createdOrder = await prisma.order.create({ ... });

// 売上明細を作成
for (const item of items) {
  await prisma.sale.create({
    data: {
      orderId: createdOrder.id,
      sku: item.sku,
      title: item.title,
      quantity: item.quantity,
      unitPrice: item.unitPrice,
      totalPrice: item.unitPrice * item.quantity,
    },
  });
}
```

### 変更内容

売上明細のforループ内で以下を追加:

1. **listing検索**: SKUまたはマーケットプレイスIDでlistingを検索し、productIdを取得
2. **Sale にlistingId/productId紐付け**: Saleレコードにも紐付ける
3. **InventoryEvent作成**: 販売イベントを記録
4. **Product.status更新**: SOLDに更新

修正後のforループ:
```typescript
// 売上明細を作成
for (const item of items) {
  // listing検索でproductId取得
  const listing = await prisma.listing.findFirst({
    where: {
      marketplace: dbMarketplace as any,
      OR: [
        { marketplaceListingId: item.sku },
        { product: { sku: item.sku } },
      ],
    },
    include: { product: true },
  });

  await prisma.sale.create({
    data: {
      orderId: createdOrder.id,
      listingId: listing?.id || null,
      productId: listing?.productId || null,
      sku: item.sku,
      title: item.title,
      quantity: item.quantity,
      unitPrice: item.unitPrice,
      totalPrice: item.unitPrice * item.quantity,
    },
  });

  // 在庫イベント（販売）+ ステータス更新
  if (listing?.productId) {
    await prisma.inventoryEvent.create({
      data: {
        productId: listing.productId,
        eventType: 'SALE',
        quantity: -(item.quantity || 1),
        prevStock: 1,
        newStock: 0,
        marketplace: dbMarketplace as any,
        orderId: createdOrder.id,
        reason: `Order synced from ${marketplace === 'joom' ? 'Joom' : 'eBay'}`,
      },
    });
    await prisma.product.update({
      where: { id: listing.productId },
      data: { status: 'SOLD' as any },
    });
  }
}
```

### 注意事項
- `dbMarketplace` は既に行207で定義されている（`'JOOM'` or `'EBAY'`）
- `prisma` は既にインポート済み
- listing検索のOR条件は、SKUがマーケットプレイスのリスティングIDの場合とプロダクトSKUの場合の両方をカバー
- 既存のコードスタイルに合わせること
- 新しいimport文は不要（Prismaのtypeは既にインポート済み）
