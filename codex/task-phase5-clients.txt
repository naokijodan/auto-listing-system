## Task: Phase 5 — Implement remaining Joom v3 API clients

Create new v3 client modules under `apps/worker/src/lib/joom/` for the following API groups.
Follow the EXACT same patterns as existing clients (orders/client.ts, products/client.ts, etc.).

### Pattern to follow:
- Each module gets its own directory: `{module-name}/client.ts`, `{module-name}/types.ts`, `{module-name}/index.ts`
- Client class extends `JoomBaseClient` from `../base-client`
- Import shared types from `../shared-types` (JoomApiResponse, JoomApiListResponse, JoomApiEmptyResponse, JoomId, Money)
- Use `this.request<T>({ method, path, query, body, accept })` pattern
- Export from module's index.ts
- Add re-exports in `apps/worker/src/lib/joom/index.ts`

### Module 1: pickup-addresses
Directory: `apps/worker/src/lib/joom/pickup-addresses/`
Class: `JoomPickupAddressesClient`

Types (types.ts):
```typescript
import { JoomId } from '../shared-types';

export interface PickupAddressContact {
  name?: string;
  phone?: string;
}

export interface PickupAddress {
  id: JoomId;
  country?: string;
  city?: string;
  streetAddress1?: string;
  streetAddress2?: string;
  zipcode?: string;
  state?: string;
  contact?: PickupAddressContact;
}

export interface CreatePickupAddressInput {
  country?: string;
  city?: string;
  streetAddress1?: string;
  streetAddress2?: string;
  zipcode?: string;
  state?: string;
  contact?: PickupAddressContact;
}

export interface UpdatePickupAddressInput {
  country?: string;
  city?: string;
  streetAddress1?: string;
  streetAddress2?: string;
  zipcode?: string;
  state?: string;
  contact?: PickupAddressContact;
}
```

Endpoints (client.ts):
- GET /pickupAddresses?id={id} → getPickupAddress(params: { id: string }): Promise<JoomApiResponse<PickupAddress>>
- GET /pickupAddresses/all → getAllPickupAddresses(): Promise<JoomApiListResponse<PickupAddress>>
- POST /pickupAddresses/create → createPickupAddress(input: CreatePickupAddressInput): Promise<JoomApiResponse<PickupAddress>>
- POST /pickupAddresses/update?id={id} → updatePickupAddress(params: { id: string }, input: UpdatePickupAddressInput): Promise<JoomApiResponse<PickupAddress>>
- POST /pickupAddresses/remove?id={id} → removePickupAddress(params: { id: string }): Promise<JoomApiEmptyResponse>
