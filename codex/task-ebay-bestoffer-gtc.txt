## タスク: Phase 1-C + 1-D - BestOffer対応 + GTC/private_listing

### 概要
eBay出品にBestOffer対応、GTC（Good 'Til Cancelled）出品期間、private_listing対応を追加する。
対象ファイル: apps/worker/src/lib/ebay-api.ts, apps/worker/src/processors/ebay-publish.ts

### 実装1: createOffer()メソッドの拡張（ebay-api.ts）

`apps/worker/src/lib/ebay-api.ts` の `createOffer()` メソッドを修正。
オファーパラメータに以下を追加:

#### offerオブジェクトの型に追加するフィールド:
```typescript
bestOfferEnabled?: boolean;
autoAcceptPrice?: number;
autoDeclinePrice?: number;
listingDuration?: string;  // 'GTC' or 'DAYS_30' etc
hideBuyerDetails?: boolean;  // private listing
```

#### createOffer() のリクエストボディに追加:
現在のcreateOffer()メソッドを読んで、リクエストボディ構築箇所を見つけること。

1. **BestOffer**: `listingPolicies` オブジェクト内に追加:
```json
{
  "listingPolicies": {
    "bestOfferTerms": {
      "bestOfferEnabled": true,
      "autoAcceptPrice": { "value": "900.00", "currency": "USD" },
      "autoDeclinePrice": { "value": "750.00", "currency": "USD" }
    }
  }
}
```
- bestOfferEnabled がtrueの場合のみ bestOfferTerms を含める
- autoAcceptPrice, autoDeclinePrice はオプション（指定された場合のみ含める）

2. **GTC（出品期間）**: リクエストボディのトップレベルに追加:
```json
{
  "listingDuration": "GTC"
}
```
- offer.listingDuration が指定されていれば使用、なければデフォルト 'GTC'

3. **Private Listing**: リクエストボディのトップレベルに追加:
```json
{
  "hideBuyerDetails": true
}
```
- offer.hideBuyerDetails が true の場合のみ含める

### 実装2: processCreateOffer()の修正（ebay-publish.ts）

`apps/worker/src/processors/ebay-publish.ts` の `processCreateOffer()` 関数（192行目付近）を修正。

createOffer() の呼び出し（226行目付近）にBestOffer/GTC/private_listingパラメータを追加:

```typescript
const result = await ebayApi.createOffer(sku, {
  marketplaceId,
  format: marketplaceData.format || 'FIXED_PRICE',
  categoryId: marketplaceData.categoryId || '',
  pricingPrice: listing.listingPrice,
  pricingCurrency: listing.currency || 'USD',
  quantity: marketplaceData.quantity || 1,
  listingDescription: marketplaceData.description || listing.product.description,
  fulfillmentPolicyId,
  paymentPolicyId,
  returnPolicyId,
  // Phase 1-C: BestOffer
  bestOfferEnabled: marketplaceData.bestOfferEnabled ?? true,  // デフォルトtrue
  autoDeclinePrice: marketplaceData.autoDeclinePrice ?? Math.floor(listing.listingPrice * 0.88),  // 88%
  // Phase 1-D: GTC + Private
  listingDuration: marketplaceData.listingDuration || 'GTC',
  hideBuyerDetails: marketplaceData.hideBuyerDetails ?? false,
});
```

### 注意事項
- createOffer() の既存のパラメータを壊さないこと
- autoDeclinePriceの計算: listingPrice * 0.88（販売価格の88%）をデフォルトとする
- autoAcceptPriceは設定しない（seller manual accept）
- listingDuration のデフォルトは 'GTC'
- hideBuyerDetails のデフォルトは false
- テストファイルがある場合はモックを更新する
- 修正後に `cd ~/Desktop/rakuda && npm test --workspace=apps/worker` で全テストが通ることを確認
