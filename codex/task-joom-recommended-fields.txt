## タスク: Phase 2-B - Joom推奨フィールド対応

### 概要
Joom出品の品質向上のため、推奨フィールドをcreateProduct()リクエストに追加する。
データソースはEnrichmentTaskのattributesとProduct。

### 対象ファイル
- apps/worker/src/lib/joom-api.ts（createProduct リクエストボディ）
- apps/worker/src/lib/joom-publish-service.ts（JoomProduct構築ロジック）

### 実装1: JoomProduct interfaceの拡張（joom-api.ts）

```typescript
interface JoomProduct {
  // 既存フィールド（変更なし）
  id?: string;
  name: string;
  description: string;
  mainImage: string;
  extraImages?: string[];
  price: number;
  currency: string;
  quantity: number;
  weight?: number;
  shipping?: {
    price: number;
    time: string;
  };
  tags?: string[];
  parentSku?: string;
  sku: string;

  // 新規追加フィールド
  brand?: string;
  categoryId?: string;       // Joom カテゴリID
  color?: string;
  size?: string;
  material?: string;
  gtin?: string;             // GTIN/JAN/EAN
  msrp?: number;             // Manufacturer's Suggested Retail Price（割引表示用）
  condition?: string;        // 'new' | 'refurbished' | 'used'
  searchTags?: string[];     // 検索タグ（tagsとは別）
  shippingLength?: number;   // cm
  shippingWidth?: number;    // cm
  shippingHeight?: number;   // cm
  dangerousKind?: string;    // 'none' | 'battery' | 'liquid' 等
}
```

### 実装2: createProduct()リクエストボディに推奨フィールド追加（joom-api.ts）

createProduct()のリクエストボディ構築部分に、productのフィールドが存在する場合のみ追加:

```typescript
// 推奨フィールド（値がある場合のみ送信）
...(product.brand ? { brand: product.brand } : {}),
...(product.categoryId ? { category_id: product.categoryId } : {}),
...(product.color ? { color: product.color } : {}),
...(product.size ? { size: product.size } : {}),
...(product.material ? { material: product.material } : {}),
...(product.gtin ? { gtin: product.gtin } : {}),
...(product.msrp ? { msrp: { amount: String(product.msrp), currency: 'USD' } } : {}),
...(product.condition ? { condition: product.condition } : {}),
...(product.searchTags?.length ? { search_tags: product.searchTags } : {}),
// Shipping dimensions（すべて揃っている場合のみ）
...(product.shippingLength && product.shippingWidth && product.shippingHeight ? {
  shipping_length: product.shippingLength,
  shipping_width: product.shippingWidth,
  shipping_height: product.shippingHeight,
} : {}),
...(product.dangerousKind ? { dangerous_kind: product.dangerousKind } : {}),
```

### 実装3: publishToJoom()でJoomProduct構築に推奨フィールド追加（joom-publish-service.ts）

publishToJoom()メソッドのJoomProduct構築部分を修正。
EnrichmentTaskのattributesとProduct情報から推奨フィールドを抽出:

```typescript
const joomProduct: JoomProduct = {
  // 既存フィールド（変更なし）
  name: translations.en.title,
  description: translations.en.description,
  mainImage: joomListing.joomImages[0] || '',
  extraImages: joomListing.joomImages.slice(1),
  price: pricing.finalPriceUsd,
  currency: 'USD',
  quantity: 1,
  weight: task.product.weight ? task.product.weight / 1000 : 0.15,
  sku: `RAKUDA-${task.productId}`,
  shipping: {
    price: pricing.shippingCost,
    time: '7-14 business days',
  },
  tags: joomCategory ? [joomCategory] : [],

  // 推奨フィールド追加
  brand: task.product.brand || attributes?.brand || undefined,
  categoryId: joomListing.joomCategory || undefined,
  color: attributes?.color || undefined,
  size: attributes?.size || undefined,
  material: attributes?.material || attributes?.caseMaterial || undefined,
  condition: 'new',  // RAKUDAは基本的に新品出品
  searchTags: attributes?.keywords || attributes?.searchTags || [],
  dangerousKind: 'none',
};
```

- `task.product` はPrismaのProductモデル
- `attributes` は `enrichmentTask.attributes as any`
- `joomListing` はJoomListingレコード
- 値がundefinedのフィールドはJoomProduct interfaceでoptionalなので問題なし

### 注意事項
- 既存の動作を壊さない（新フィールドはすべてオプショナル）
- undefinedのフィールドはcreateProduct()のリクエストボディに含まれない（スプレッド演算子で条件付き追加）
- MSRP は pricing.costUsd * 2 程度（仕入れ値の2倍）をデフォルトにしない。明示的に設定されている場合のみ送信
- conditionは 'new' をデフォルトにする（RAKUDAは新品出品が主）
- dangerousKindは 'none' をデフォルトにする（時計は危険物ではない）
- テストが壊れないようにする
