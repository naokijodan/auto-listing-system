## タスク: eBay出品UI E2Eテスト追加（BestOffer/GTC/ポリシー選択）

### 目的
eBay出品テンプレートページの新機能（BestOffer設定、GTC出品期間選択、DBからのビジネスポリシー選択）に対するPlaywright E2Eテストを追加する。

### 作成するファイル
`apps/web/e2e/ebay-template-settings.spec.ts`

### テスト対象ページ
- URL: `/ebay/templates`
- ファイル: `apps/web/src/app/ebay/templates/page.tsx`

### テストケース

#### 1. ページ表示テスト
- テンプレート管理ページが正しく表示されること
- ヘッダーにeBayテンプレート関連のテキストが表示されること

#### 2. Best Offer設定テスト
- Best Offer有効化トグル（Switch）が表示されること
- トグルをONにするとBestOfferが有効になること
- 自動承認価格（USD）入力フィールドが表示されること
- 最低受付価格（USD）入力フィールドが表示されること
- 価格入力が可能なこと

#### 3. GTC（出品期間）選択テスト
- 出品期間のSelectコンポーネントが表示されること
- デフォルト値が "Good 'Til Cancelled" であること
- ドロップダウンを開くと6つのオプション（GTC, 30日, 10日, 7日, 5日, 3日）が表示されること
- オプションを選択できること

#### 4. ビジネスポリシー選択テスト
- 配送ポリシー（Fulfillment）のSelectが表示されること
- 支払いポリシー（Payment）のSelectが表示されること
- 返品ポリシー（Return）のSelectが表示されること
- ポリシーデータ読み込み中に「読み込み中...」と表示されること（ローディング状態）

#### 5. テンプレート保存テスト
- テンプレート名を入力して保存ボタンをクリックできること
- 保存後にテンプレート一覧に表示されること（可能であれば）

### 既存E2Eテストのパターン（参考）

ファイル: `apps/web/e2e/ebay-publish-flow.spec.ts`
```typescript
import { test, expect } from '@playwright/test';

const API_BASE = process.env.API_BASE || 'http://localhost:3010';

test.describe('eBay Template Settings', () => {
  test('should display template page', async ({ page }) => {
    await page.goto('/ebay/templates');
    await page.waitForTimeout(1500);
    await expect(page.locator('body')).toBeVisible();
    // ヘッダー確認
    const heading = page.getByRole('heading', { name: /テンプレート|template/i });
    await expect(heading).toBeVisible();
  });
});
```

### Playwright設定
- テストディレクトリ: `apps/web/e2e/`
- baseURL: `http://localhost:3012`（playwright.config.tsの設定）
- テスト実行: `cd apps/web && npx playwright test e2e/ebay-template-settings.spec.ts`

### UIコンポーネントの具体的なロケーター情報

#### BestOffer関連（テンプレートページ内）
- Switch: `id="bestOfferEnabled"`
- 自動承認価格: `id="autoAcceptPrice"` (type=number)
- 最低受付価格: `id="minimumBestOfferPrice"` (type=number)

#### GTC（出品期間）
- Select: ラベルは「出品期間」
- SelectValue placeholder: "出品期間を選択"
- オプション: GTC, Days_30, Days_10, Days_7, Days_5, Days_3

#### ポリシー選択
- 配送ポリシー: `id="fulfillmentPolicyId"` placeholder="配送ポリシーを選択"
- 支払いポリシー: `id="paymentPolicyId"` placeholder="支払いポリシーを選択"
- 返品ポリシー: `id="returnPolicyId"` placeholder="返品ポリシーを選択"
- データはSWR経由で `/api/ebay/policies?marketplaceId=EBAY_US` から取得

### 注意事項
- テストはローカルサーバー起動前提（port 3012）
- APIサーバーがport 3010で起動していること
- DB接続が必要（ポリシーデータ取得のため）
- テストが完了したら全E2Eテストが通ることを確認: `cd apps/web && npx playwright test`
