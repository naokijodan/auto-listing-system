## タスク: Phase 2-C - Joom国別送料設定

### 概要
Joom出品の送料を改善。国別送料テーブルを追加し、
Joom Logistics vs Offline Shippingの切り替えを可能にする。

### 対象ファイル
- apps/worker/src/lib/joom-publish-service.ts（送料計算ロジック）
- apps/worker/src/lib/joom-api.ts（createProductリクエストに送料情報追加）

### 実装1: 送料テーブル定数（joom-publish-service.ts）

ファイルの先頭付近（import文の後）に以下の送料設定を追加:

```typescript
/**
 * Joom国別送料テーブル（USD）
 * 重量ベース: base + (weightKg * perKg)
 */
const JOOM_SHIPPING_RATES: Record<string, { base: number; perKg: number }> = {
  // Tier 1: 主要市場（Joom Logistics対応）
  US: { base: 5.00, perKg: 2.00 },
  DE: { base: 4.50, perKg: 1.80 },
  FR: { base: 4.50, perKg: 1.80 },
  IT: { base: 5.00, perKg: 2.00 },
  ES: { base: 5.00, perKg: 2.00 },
  GB: { base: 5.50, perKg: 2.20 },
  // Tier 2: CIS市場
  RU: { base: 3.00, perKg: 1.50 },
  BY: { base: 3.50, perKg: 1.80 },
  KZ: { base: 4.00, perKg: 2.00 },
  // Tier 3: その他
  DEFAULT: { base: 6.00, perKg: 2.50 },
};

/**
 * 送料計算
 */
function calculateShippingCost(weightKg: number, countryCode: string = 'DEFAULT'): number {
  const rate = JOOM_SHIPPING_RATES[countryCode] || JOOM_SHIPPING_RATES.DEFAULT;
  return Math.round((rate.base + weightKg * rate.perKg) * 100) / 100;
}

/**
 * 送料方式
 */
type JoomShippingMethod = 'joom_logistics' | 'offline';

/**
 * Joom Logistics対応国
 */
const JOOM_LOGISTICS_COUNTRIES = ['US', 'DE', 'FR', 'IT', 'ES', 'GB'];
```

### 実装2: publishToJoom()で送料計算ロジック更新（joom-publish-service.ts）

publishToJoom()メソッドの送料計算部分を修正。
現在はpricing.shippingCostを使っているが、国別送料を使うように変更:

```typescript
// 現在:
// shipping: { price: pricing.shippingCost, time: '7-14 business days' }

// 修正後:
const weightKg = task.product.weight ? task.product.weight / 1000 : 0.15;
const defaultShipping = calculateShippingCost(weightKg);

// JoomProductのshippingフィールド
shipping: {
  price: defaultShipping,
  time: '7-14 business days',
},
```

### 実装3: createProduct()にshipping_method追加（joom-api.ts）

createProduct()のリクエストボディに送料方式を追加:

JoomProduct interfaceに追加:
```typescript
shippingMethod?: 'joom_logistics' | 'offline';
```

createProduct()のリクエストボディに追加（variantsの前あたり）:
```typescript
...(product.shippingMethod ? { shipping_method: product.shippingMethod } : { shipping_method: 'offline' }),
```

publishToJoom()のJoomProduct構築に追加:
```typescript
shippingMethod: 'offline',  // デフォルトはoffline（Joom Logistics未契約のため）
```

### 注意事項
- Joom Logistics は契約が必要なので、デフォルトは 'offline'
- 国別送料はリクエストボディではなくpricing計算に反映（Joom APIは単一の送料値のみ受け付ける）
- shippingCostのデフォルト値は、DEFAULT レートで計算した値を使用
- JoomProduct.shippingの型は変更しない
- テストが壊れないようにする
- 修正後に `cd ~/Desktop/rakuda && npm test --workspace=apps/worker` で全テストが通ることを確認
