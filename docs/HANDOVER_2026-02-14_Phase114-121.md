## 新規Phase詳細

### Phase 120: 多言語対応

**サポート言語（12言語）**:
- 英語(en), ドイツ語(de), フランス語(fr), スペイン語(es)
- イタリア語(it), 日本語(ja), 中国語(zh), 韓国語(ko)
- ポルトガル語(pt), オランダ語(nl), ポーランド語(pl), ロシア語(ru)

**データ保存**:
```json
// listing.marketplaceData
{
  "translations": {
    "de": { "title": "...", "description": "..." },
    "fr": { "title": "...", "description": "..." }
  },
  "lastTranslatedAt": "2026-02-14T..."
}
```

### Phase 121: プロモーション連携

**プロモーションタイプ**:
- `MARKDOWN_SALE` - マークダウンセール
- `ORDER_DISCOUNT` - 注文割引
- `VOLUME_PRICING` - ボリューム割引
- `SHIPPING_DISCOUNT` - 送料割引
- `COUPON` - クーポン
- `CODELESS_COUPON` - コードレスクーポン

**テンプレート**:
- フラッシュセール（24時間20%OFF）
- 週末セール（48時間15%OFF）
- まとめ買い割引（3点以上10%OFF）
- 送料無料（$50以上）
- 新規顧客割引（10%OFF）
