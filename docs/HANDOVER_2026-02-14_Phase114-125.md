## Phase 125 詳細: メッセージ自動応答

### トリガータイプ
| タイプ | 名前 | 説明 |
|--------|------|------|
| `NEW_MESSAGE` | 新規メッセージ受信 | バイヤーからの新規メッセージ |
| `ORDER_INQUIRY` | 注文に関する問い合わせ | 注文状況・追跡の問い合わせ |
| `SHIPPING_INQUIRY` | 配送に関する問い合わせ | 配送日・配送方法の質問 |
| `RETURN_REQUEST` | 返品リクエスト | 返品・返金の要求 |
| `PRODUCT_QUESTION` | 商品に関する質問 | 商品詳細・仕様の質問 |
| `OFFER_RECEIVED` | オファー受信 | 価格交渉オファー |
| `NEGATIVE_FEEDBACK` | ネガティブフィードバック | 低評価・クレーム |
| `OUT_OF_OFFICE` | 営業時間外 | 営業時間外の自動応答 |

### 応答モード
| モード | 名前 | 説明 |
|--------|------|------|
| `TEMPLATE` | テンプレート | 定型文を使用 |
| `AI_GENERATED` | AI生成 | AIが文脈に応じた返信を生成（GPT-4o） |
| `HYBRID` | ハイブリッド | テンプレート＋AI補完 |

### AI分析機能
- **インテント検出**: メッセージの意図を自動分類
- **センチメント分析**: ポジティブ/ネガティブ/ニュートラル
- **優先度判定**: LOW/MEDIUM/HIGH/URGENT
- **キーワード抽出**: 重要なキーワードを抽出
